## Features

* Allow users to create a collection to store favorite links
* Allow users to set collection visibility public/private

  * All those with public visibility, user will be able to share given links or integrate it (avaiable as JSON / Normal embedded page)
  * Public visibilty will be show on the front pages with other users
* When storing given link - user can add brief description, tags (select new or existing one)
* Datalist filtering by Date, (most click)
* Will add an analytic storage to see which one click most
* Allow users to create a group collection with other user (optional)
* Can access through browser extensions.
